Gym Backend API

A RESTful backend for a fitness tracking application built with Laravel.
The system allows users to manage workouts, log training sessions, track progress, and create diet plans.

⸻

Features

• User Authentication (Laravel Sanctum)
• Workout Management
• Workout Logging
• Workout Progress Tracking
• Workout Plans
• Diet Plans
• RESTful API structure

⸻

Tech Stack

• Laravel
• Laravel Sanctum
• SQLite
• REST API
